# ChippAPI
-> This is a simple python wrapper for [Chipp API](https://chipp.ai/).

## Installation
```bash
pip install chippapi
```

## Usage
```python
import chippapi

chippapi.api_key = "YOUR_API_KEY"
chippapi.id = "YOUR_ID"

response = chippapi.chat(prompt="Hello, how are you?")
print(response)
```
